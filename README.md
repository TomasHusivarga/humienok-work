# Humienok work

Repo pre zdielane Humienok subory.

## Pre firmy newsletter

Najlepsie pouzit tento copy tool:

- https://raw.githack.com/TomasHusivarga/humienok-work/main/copy-newsletter.html?v=4

Toto pouzivaj, ked treba vlozit hotovy newsletter do mailoveho editora. Nekopirovat z Gmailu a neposielat cez preposlany mail, lebo potom sa tam mozu dostat hlavicky spravy alebo sa stratia obrazky/tlacidla.

Postup:

1. Otvor link vyssie.
2. Klikni `Skopirovat newsletter`.
3. Ak sa prehliadac spyta na povolenie schranky, povol to.
4. V mailovom editore vloz cez `Ctrl+V`.

Len nahlad:

- https://raw.githack.com/TomasHusivarga/humienok-work/main/newsletters/humienok-pre-firmy-newsletter.html

Zdrojovy subor:

- `newsletters/humienok-pre-firmy-newsletter.html`

Obrazky sa nacitavaju z `https://www.humienok.sk/wp-content/uploads/2026/06/`, takze v mailoch nemusia byt lokalne subory.
