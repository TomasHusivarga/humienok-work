# Humienok work

This repository stores shareable Humienok work files.

## Pre firmy newsletter

File:

- `newsletters/humienok-pre-firmy-newsletter.html`

How to use it:

1. Open the HTML file in GitHub.
2. Click **Raw**.
3. Copy the full HTML code.
4. Paste it into the email/newsletter tool that accepts custom HTML.

The newsletter images are loaded from `https://www.humienok.sk/wp-content/uploads/2026/06/`, so they should display inside emails without needing local image files.
