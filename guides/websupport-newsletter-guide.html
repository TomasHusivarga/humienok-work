Ako vlozit newsletter vo WebSupport webmaili

Do bieleho pola to nevkladaj ako normalny text. Webmail potom rozhadze dizajn. Treba pouzit tlacidlo <> a vlozit HTML kod do zdrojoveho kodu.

Najprv skopiruj HTML kod

Otvor copy tool, rozklikni Zaloha: HTML kod, oznac cely kod a skopiruj ho cez Ctrl+C.

Link na copy tool: copy-newsletter.html

1

Otvor novy e-mail

Vypln adresata a predmet. Do bieleho tela e-mailu este nic nevkladaj.

BIlink<>undo
Prazdne telo e-mailu
2

Klikni na tlacidlo <>

Toto otvori okno Source Code. Tam patri HTML kod newslettera.

obrazok<>lupa
Klikni prave na tuto ikonu <>, nie do bieleho pola.
3

Vloz HTML kod

V okne Source Code najprv vymaz vsetko, co tam je. Potom vloz kod cez Ctrl+V.

<table role="presentation" width="100%" ...>
  <tr>
    <td>... newsletter ...</td>
  </tr>
</table>
Potom klikni Save / Ulozit.
4

Skontroluj nahlad a posli test

Po ulozeni by sa mal newsletter zobrazit v tele mailu. Najprv ho posli sebe ako test, az potom klientom.

ZlePastnut do bieleho pola ako normalny text. Webmail rozhadze obrazky, farby a rozlozenie.
DobrePastnut HTML do <> Source Code okna a ulozit.

Ked sa to aj tak rozbije

Potom WebSupport webmail tento typ HTML newslettera obmedzuje. Vtedy je lepsie pouzit Ecomail, Brevo, Mailchimp alebo iny newsletterovy nastroj.
